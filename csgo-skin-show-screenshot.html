去灰抠图、透明手臂，CSGO皮肤截图（玩家秀）教程&指令

2022-02-07 分类：CSGO指令命令 / CSGO教程攻略 阅读(7538) 评论(0)

本文将介绍玩家的CSGO皮肤截图，即玩家秀常见的制作方法，比如模糊背景+清晰皮肤，包括需要用到的指令，另外还有透明手臂的实现方法，可用于截取皮肤全身截图。注意，需要用到Photoshop工具。

具体内容请看视频：

https://www.ixigua.com/7061854814900224516

必备指令cfg

进入地图后使用，包括：作弊模式、回合时间60分钟（各模式）、关闭胜利、踢出BOT、重开游戏等。

链接：https://pan.baidu.com/s/12R9-Qlub-F_M2muc9xK6Sg?pwd=nobf
提取码：nobf

exec rushbnet

专用截图地图

Screenshot Map by Nitrate (62 in 1)

https://steamcommunity.com/sharedfiles/filedetails/?id=916771218

关闭HUD界面

cl_drawhud 0

关闭黑雾效果/灰尘

mat_postprocess_enable 0

隐藏武器和手臂

r_drawviewmodel 0

透明手臂

需要先下载CFG文件和vscripts文件：

链接：https://pan.baidu.com/s/1KShKpQ92XyZ1FKS6nd7GGQ?pwd=rush
提取码：rush

有两个文件，CFG文件放置到\Counter-Strike Global Offensive\csgo\cfg文件夹里，nut文件（vscripts）放置到\Counter-Strike Global Offensive\csgo\scripts\vscripts文件夹里。

准备完成后，输入以下指令可实现透明手臂：

exec nohands

视角固定

cl_lock_camera 1

飞天

noclip

武器向左向右旋转角度

setang 0 0 角度
