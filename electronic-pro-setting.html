electronic鼠标键盘外设、视频设置、准心CFG参数

2019-08-26 分类：选手外设/设置 阅读(26517) 评论(0)

以下是Natus Vincere俱乐部CSGO职业选手Denis “electronic” Sharipov（外号电子哥）所使用的鼠标、键盘、显示器、耳机等硬件外设、以及视频画面设置、准心（准星）和视角指令参数等：

配件外设

  • 显示器：卓威奇亚 XL2540 （购买地址）
  • 鼠标：罗技（G）G703 （购买地址）
  • 鼠标垫：金士顿(Kingston) HyperX 复仇之焰（购买地址）
  • 键盘：金士顿(Kingston) HyperX 阿洛伊 FPS RGB（购买地址）
  • 耳机：金士顿 HyperX Cloud II（购买地址）
  • 显卡：英伟达GEFORCE RTX 1080 TI（购买地址）

鼠标设置

  • DPI：400
  • 灵敏度：2.20
  • eDPI：880
  • HZ：1000
  • 开镜时灵敏度：1.00
  • 鼠标加速：关
  • 系统灵敏度：6
  • 原始输入：关

视频设置

  • 画面分辨率：1280×960
  • 宽高比：4:3
  • 缩放模式：拉伸
  • HZ：240

准心（准星）参数

cl_crosshaircolor 1; cl_crosshairdot 0; cl_crosshairgap -3; cl_crosshairsize 2.5; cl_crosshairstyle 4; cl_crosshairthickness 0; cl_crosshair_drawoutline 0; cl_crosshairalpha 200; cl_crosshaircolor_b 250; cl_crosshaircolor_g 250; cl_crosshaircolor_r 250;

视角设置

viewmodel_fov 68; viewmodel_offset_x 2.5; viewmodel_offset_y 0; viewmodel_offset_z -1.5; viewmodel_presetpos 3; cl_viewmodel_shift_left_amt 1.5; cl_viewmodel_shift_right_amt 0.75; viewmodel_recoil 0; cl_righthand 1;

视角（手臂/武器）晃动

cl_bob_lower_amt 5; cl_bobamt_lat 0.33; cl_bobamt_vert 0.14; cl_bobcycle 0.98;
