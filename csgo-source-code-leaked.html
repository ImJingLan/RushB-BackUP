关于CSGO源代码泄露官方公告

2020-04-23 分类：CSGO新闻资讯 阅读(3453) 评论(0)

在今日凌晨，各大游戏媒体报道了有关V社旗下游戏反恐精英：全球攻势（CSGO）和军团要塞2（TF2）游戏源代码泄露，让黑客更容易开发外挂以破坏游戏环境。不过最新官方公告表示，这些泄露并不会对游戏造成影响。

由于游戏社区议论纷纷，CSGO团队已经发布了公告：

我们（CSGO团队）已经审查了泄露的代码，认为这些是2017年年末提供给合作伙伴的部分CSGO引擎代码库，并在2018年泄露。通过审查，我们认为这次事件并不足以让玩家惊慌或对当前游戏版本造成影响。

如果我们发现新的证据，会继续调查情况，并更新公告给玩家。同时，如果有人对此次泄露有更多的了解，欢迎在Valve安全页面报告：https://www.valvesoftware.com/en/security
