Shox鼠标键盘外设、视频设置、准心CFG参数

2019-12-16 分类：选手外设/设置 阅读(14674) 评论(0)

以下是Vitality俱乐部CSGO职业选手Richard “shox” Papillon所使用的鼠标、键盘、显示器、耳机等硬件外设、以及视频画面设置、准心（准星）和视角控制台指令参数等：

配件外设

  • 显示器：ZOWIE GEAR 卓威奇亚 XL2546（购买地址）
  • 鼠标：ZOWIE GEAR 卓威 奇亚 FK1（购买地址）
  • 鼠标垫：ZOWIE GEAR 卓威 奇亚 G-SR（购买地址）
  • 键盘：Xtrfy K2 （购买地址）
  • 耳机：罗技（G）PRO X（购买地址）

鼠标设置

  • DPI：400
  • 灵敏度：1.80
  • eDPI：720
  • HZ：1000
  • 开镜时灵敏度：1.00
  • 鼠标加速：关
  • 系统灵敏度：6
  • 原始输入：开

视频设置

  • 画面分辨率：1024×768
  • 宽高比：4:3
  • 缩放模式：黑边
  • HZ：240

准心（准星）参数

cl_crosshair_drawoutline 0; cl_crosshair_sniper_width 1; cl_crosshair_t 0; cl_crosshairalpha 5000; cl_crosshaircolor 2; cl_crosshaircolor_b 0; cl_crosshaircolor_g 255; cl_crosshaircolor_r 255; cl_crosshairdot 0; cl_crosshairgap -2; cl_crosshairgap_useweaponvalue 0; cl_crosshairsize 2; cl_crosshairstyle 2; cl_crosshairthickness 1.15; cl_crosshairusealpha 1;

视角设置

viewmodel_fov 68; viewmodel_offset_x 2.5; viewmodel_offset_y 0; viewmodel_offset_z -1.5; viewmodel_presetpos 3; cl_viewmodel_shift_left_amt 1.5; cl_viewmodel_shift_right_amt 0.75; viewmodel_recoil 0; cl_righthand 1;

视角（手臂/武器）晃动

cl_bob_lower_amt 5; cl_bobamt_lat 0.33; cl_bobamt_vert 0.14; cl_bobcycle 0.98;
