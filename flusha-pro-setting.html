flusha鼠标键盘外设、视频设置、准心CFG参数

2019-10-26 分类：选手外设/设置 阅读(7897) 评论(0)

以下是Fnatic俱乐部CSGO职业选手Robin “flusha” Rönnquist（外号Senor Vac）所使用的鼠标、键盘、显示器、耳机等硬件外设、以及视频画面设置、准心（准星）和视角控制台指令参数等：

配件外设

  • 显示器：ZOWIE GEAR 卓威奇亚 XL2540（购买地址）
  • 鼠标：罗技（G）G703 LIGHTSPEED（购买地址）
  • 鼠标垫：FNATIC GEAR FOCUS 2
  • 键盘：FNATIC GEAR RUSH G1
  • 耳机：金士顿（Kingston） HYPERX CLOUD ALPHA（购买地址）
  • 显卡：华硕 GTX 1080 TI （购买地址）

鼠标设置

  • DPI：400
  • 灵敏度：1.70
  • eDPI：680
  • HZ：1000
  • 开镜时灵敏度：0.87
  • 鼠标加速：关
  • 系统灵敏度：6
  • 原始输入：开

视频设置

  • 画面分辨率：1280×960
  • 宽高比：4:3
  • 缩放模式：拉伸
  • HZ：240

准心（准星）参数

cl_crosshairalpha 999; cl_crosshaircolor 0; cl_crosshairdot 1; cl_crosshairgap -4; cl_crosshairsize 0; cl_crosshairstyle 4; cl_crosshairusealpha 1; cl_crosshairthickness 0.5; cl_crosshair_drawoutline 1; cl_crosshair_sniper_width 1;

视角设置

viewmodel_fov 68; viewmodel_offset_x 2.5; viewmodel_offset_y 0; viewmodel_offset_z -1.5; viewmodel_presetpos 3; cl_viewmodel_shift_left_amt 1.5; cl_viewmodel_shift_right_amt 0.75; viewmodel_recoil 1; cl_righthand 1;

视角（手臂/武器）晃动

cl_bob_lower_amt 21; cl_bobamt_lat 0.4; cl_bobamt_vert 0.25; cl_bobcycle 0.98;
